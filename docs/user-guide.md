## Getting Started

### System Requirements
- **For Owners:** Web browser (Chrome, Firefox, Safari, Edge)
- **For Drivers:** WhatsApp on mobile phone
- **Internet Connection:** Stable internet connection required

### Initial Setup
1. Contact your system administrator for account creation
2. Receive login credentials via SMS/Email
3. For drivers: Save the WhatsApp bot number in your contacts

## For Truck Owners

### Accessing the Dashboard
1. Open your web browser
2. Go to the platform URL provided by your administrator
3. Enter your username and password
4. Click "Login"

### Dashboard Overview
The main dashboard shows:
- **Active Trips:** Currently running trips
- **Available Drivers:** Drivers ready for new assignments
- **Financial Summary:** Today's income and expenses
- **Recent Activities:** Latest system updates

### Managing Drivers

#### Adding a New Driver
1. Go to "Drivers" section
2. Click "Add Driver"
3. Fill in driver details:
   - Name
   - Phone number (with WhatsApp)
   - License number
   - Assigned vehicle
4. Click "Save"

#### Checking Driver Availability
- Green dot: Driver is available
- Red dot: Driver is busy
- Gray dot: Driver is offline

## For Drivers

### Getting Started with WhatsApp
1. Save the bot number provided by your owner
2. Send "START" to register with the system
3. Follow the setup instructions

### Basic Commands

#### Marking Availability
- Send "FREE" when you're available for new trips
- Send "BUSY" when you're occupied
- Send "OFFLINE" when you're not working

#### Trip Management
- You'll receive trip offers via WhatsApp
- Reply "YES" to accept a trip
- Reply "NO" to decline

#### During the Trip

**At Pickup:**
- Send "PICKED UP" when you collect the load
- Take a photo of the loading receipt and send it

**During Transit:**
- Send location updates: "AT [CITY NAME]"
- Log fuel expenses: "FUEL 5000" (amount in rupees)
- Log other expenses: "EXPENSE 500 Toll tax"

**At Delivery:**
- Send "DELIVERED" when you complete delivery
- Take a photo of the delivery receipt and send it

## Troubleshooting

### Common Issues for Owners

#### Cannot Login
- Check username/password
- Ensure internet connection
- Clear browser cache
- Contact support if issue persists

#### Dashboard Not Loading
- Refresh the page
- Try different browser
- Check internet connection
- Wait a few minutes and retry

#### Driver Not Responding
- Check if driver's phone has internet
- Verify WhatsApp number is correct
- Call driver directly if urgent

### Common Issues for Drivers

#### Bot Not Responding
- Check WhatsApp internet connection
- Verify you're messaging the correct bot number
- Restart WhatsApp app
- Contact your owner

#### Commands Not Working
- Use exact commands as specified
- Check spelling and format
- Don't add extra words
- Send one command per message

#### Photo Not Accepted
- Take clear, well-lit photos
- Ensure text is readable
- Check file size (max 5MB)
- Try taking photo again
